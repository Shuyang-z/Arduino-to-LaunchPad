# Arduino-to-LaunchPad
Arduino Nano to MSP430F5529/MSP432P401R LaunchPad adapter board for TREEE_TEA1 robot

Pins definition:

|  Functioon  |  Arduino Nano  |  LaunchPad  |
|-------------------- UART ------------------|
|     TX      |        D1      |      4      |
|     RX      |        D0      |      3      |
|-------------------- PWM -------------------|
|  PWM_LEFT   |        D6      |     40      |
|  PWM_RIGHT  |        D9      |     39      |
|  DIR_LEFT   |        D7      |     38      |
|  DIR_RIGHT  |        D10     |     37      |
|-------------------- TACH ------------------|
|  TACH_LEFT  |        D2      |     36      |
|  TACH_RIGHT |        D3      |     35      |
|-------------------- LED -------------------|
|     LED     |        D4      |      5      |
|------------------ 74HC165 -----------------|
|     SL      |        D8      |      8      |
|    MISO     |        D12     |     14      |
|    SCK      |        D13     |      7      |
